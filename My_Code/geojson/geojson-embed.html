To embed the geojson map in html:

Change the format from:
https://github.com/WendyAnthony/Code_Each_Day/blob/master/My_Code/geojson/Mapping-Upriver-Sto%CC%81lo%CC%84-Ethnography-trial-3.geojson
to
https://embed.github.com/view/geojson/WendyAnthony/Code_Each_Day/master/My_Code/geojson/Mapping-Upriver-Sto%CC%81lo%CC%84-Ethnography-trial-3.geojson

By adding {embed} at beginning,
{view/geojson} after github.com
remove {blob/}
and insert within a script tag

The embedded map is 420px x 620px by default - customize output by adding height and width variables as parameters at the end, within the script src quotation marks, e.g. ?height=300&width=500.
