About This App

An experimental Shiny Code Sandbox was created to develop an online, interactive interface to facilitate Geography Course Student Planning.
This Shiny Clone Website is meant to mimic the Shiny App on the web, without relying on a Shiny Server.

Goals:

  • to make a web page javascript version of the Geog Course Datatable as in Shiny App
  • Make the table interactive, sortable filterable, saveable
