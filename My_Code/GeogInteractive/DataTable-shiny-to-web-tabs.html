GEOG 100 Geography for Today Each topic stretches the geographical imagination by offering a new lens through which to view the relationship between people and the richly diverse and varied environments they inhabit. About This App

Goals | Workflow | Accessability | The Code | Change History

An experimental Shiny Code Sandbox was created to develop an online, interactive interface to facilitate Geography Course Student Planning.
This Shiny Clone Website is meant to mimic the Shiny App on the web, without relying on a Shiny Server.

Goals:

  • to make a web page javascript version of the Geog Course Datatable as in Shiny App
  • Make the table interactive, sortable filterable, saveable

Menu

Workflow:

  • HTML Table Code:
    • Opened Shiny App Datatable, changing to Show: 100 which will open whole table
    • In Chrome Browser Develoment Tools >> Elements >> Find `table` element
      • Hover over code to see table in webpage highlighted >>
      • Right-click `Table` Element in Elements list >> `Edit as HTML` >>
      • Right-click inside `Table` Element >> Select all >> Copy >> Paste into html file
    • Tried originally using `CsvToTable.js`, but it got confused with commas

Menu

Accessability Tips

  • Accessible Keystrokes
    • After opening webpage >> Tab >> Highlights 1st Tab >> Right-Arrow >> Highlights 2nd Tab >> Right-Arrow keeps scrolling through top level tabs
    • Tab again >> moves through 2nd level tabs >> Enter opens Tab >>
    • After reaching end of 2nd level tabs, Tab again moves through tab body highlighting links >> Enter opens link >> after tabbing through all links >> highlights About tab again
    • From top level tab About >> Tab once highights tab body >>
    • Left-Arrow key moves backwards through 1st level tabs
